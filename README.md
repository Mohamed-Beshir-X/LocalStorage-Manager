## Description
LocalStorage Manager is a simple web app that demonstrates the power of browser Local Storage. 
It allows you to:
- Add new items to Local Storage
- Remove items
- Check if a specific item exists
- Retrieve and display all stored items with a single click

This project is built with vanilla HTML, CSS, and JavaScript. It's a simple, interactive tool to practice data management in the browser.
